connect-papers

Scraping ResearchGate for citation-relationships and visualizing them

Python Webscraping Selenium Webdriver pyvis

Development Period: 08.2023

While doing research for the Performance isolation paper, I was inspired to make a tool for visualizing references between the research papers. After implementing the first prototype I found out there are already plenty of advanced commercial solutions supporting this. ðŸ˜…

To achieve this, I implemented a ResearchGate scraper that would take a list of papers and build a set of their citation-relationships as well as extend these with citations up to depth d. These relationships were then visualized in form of an interactive undirected graph.

Code available in the GitHub repository.
